## Glossary

- **Daemon**: A background process that starts automatically and runs continuously.
- **systemd**: The Linux service manager (Ubuntu).
- **Cooldown**: The minimum wait time between recovery actions to avoid infinite loops.
- **Rate limiting**: Limiting the frequency of notifications.
- **PID file**: A file that stores the daemon process ID.
- **CPU Load**: The average system load over 1/5/15 minutes.
- **Mount point**: The attachment point of a partition or disk.
