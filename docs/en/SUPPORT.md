## Support channels

- Email: license@xnetvn.net

## Bug reports and feature requests

Please use the templates in `.github/ISSUE_TEMPLATE/` or `docs/en/templates/` to
ensure complete information.

## Information required for support requests

- Version (`__version__`).
- Operating system and kernel.
- Related configuration (mask sensitive data).
- Error logs (`journalctl -u xnetvn_monitord -f`).
